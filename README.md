# Swegg

--------------------

### Basic setup
```{bash}
cd msconnect
virtualenv --python=python3 venv
source venv/bin/activate
pip install -r requirements.txt
```

### Local server for development
```{bash}
source venv/bin/activate
python run.py
```
Now point your browser to **http://localhost:5000**
